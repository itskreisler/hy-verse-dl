# hy-verse-dl

Una version alternativa :)



## Ejecutar localmente

Clone el proyecto

```bash
  git clone https://github.com/itskreisler/hy-verse-dl.git
```

Ir al directorio del proyecto

```bash
  cd hy-verse-dl
```

Instalar dependencias

```bash
  npm install
```

Iniciar el servidor

```bash
  npm run dev
```
